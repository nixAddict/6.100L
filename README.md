# MIT 6.100L - Introduction to CS and Programming using Python

Coursework for MIT's 6.100L, a full-semester introduction to computer science and programming using Python. Aimed at students with little to no programming experience.

## Problem Sets

| Problem Set | Topic | Description |
|-------------|-------|-------------|
| **PS0** | Getting Started | Python setup and basic operations |
| **PS1** | Saving Money | Calculate months to save for house down payment using bisection search |
| **PS2** | Hangman | Classic word-guessing game implementation |
| **PS3** | Document Distance | Calculate similarity between text documents |
| **PS4** | Word Games | Scrabble-like game with scoring and substitution ciphers |
| **PS5** | Image Processing | Image manipulation, filters, and steganography |

## Quick Start

```bash
# Clone the repository
git clone https://github.com/nixAddict/6.100L.git
cd 6.100L

# Run any problem set
cd ps1
python3 ps1a.py

# Run tests
python3 ps1_tester.py
```
